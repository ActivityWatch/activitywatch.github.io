Nightly builds

Unstable "nightly" builds are automated by our CI setup. These contain yet-unreleased features and bug fixes. See issue #507 for where to get them.

Watchers

What is a watcher? A watcher is a program or plugin that watches for activity of different kinds and reports it to ActivityWatch.

ActivityWatch comes with two watchers preinstalled:

  • Window watcher: tracks the active application/window
  • AFK watcher: tracks if you're using your mouse and keyboard to determine if you are AFK.
The documentation contains a list of all available watchers (editor plugins, for example).

Web browser watchers

Use these web extensions to get more detailed tracking about the websites you visit (such as the URL, or if a tab is playing sound).
